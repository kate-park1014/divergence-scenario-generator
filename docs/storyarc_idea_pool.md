# Storyarc 아이디어 풀

같은 보스 데이터라도 NPC/세계관을 바꾸면 완전히 다른 서사가 된다. 이 문서는 "비운의 얼음 계승자 = 스스로 격리한 여왕" 같은 **공통 보스 컨셉**에 대해 가능한 여러 서사 축을 기록한다. 다른 보스 타입에도 응용 가능.

---

## Case Study: 아스트리엘 (폭주하는 얼음의 계승자) — 5가지 서사 축

보스 핵심 설정: 힘을 통제 못하는 비운의 계승자 / 사랑하는 이를 다치게 할까 봐 스스로 얼음 성에 격리 / 폭풍은 공격이 아닌 "다가오지 마"라는 거절.

### 1. 자매 재회 (Frozen 스타일 — 기본값, 지양 권장)
- **NPC**: 잃어버린 여동생 / 혈연.
- **서사 축**: "언니, 나야." 가족 재회.
- **주의**: 겨울왕국 연상이 너무 강함. 다른 축이 독창성 높음.

### 2. 전설 추적자 (조사극)
- **NPC**: 사라진 왕조의 공주 전설을 쫓는 기록자/학자. 사적 관계 없음.
- **서사 축**: 단서 조합으로 "겨울 군주 = 잊힌 계승자"임을 밝혀가는 조용한 조사극.
- **톤**: 발견 / 역사적 증명. 재회가 아니라 이름을 복원하는 이야기.
- **복선 예**: 왕조 문장, 깨진 인장, 지워진 이름, 잊힌 대관식 기록.
- **클라이맥스**: 탐험대가 그녀의 이름을 불러주는 순간 — 감정이 아니라 기억이 녹인다.

### 3. 청부 의뢰와 양심 (도덕 딜레마)
- **NPC**: 얼어붙은 마을의 유족에게 "처치" 계약을 받고 온 청부업자. 혹은 그 계약을 끼워넣은 동료.
- **서사 축**: 계약을 수행하러 왔다가 진실(그녀가 의도한 게 아니었다)을 알고 도덕적 붕괴.
- **톤**: 하드보일드, 회색. 해피엔딩도 비극도 가능.
- **복선 예**: 계약서, 의뢰인의 초상화, 무기에 맺히는 서리, 얼어붙은 마을 사람의 마지막 표정.
- **클라이맥스**: 칼을 내려놓을 것인가, 계약을 이행할 것인가. 선택의 무게.

### 4. 옛 동문의 죄책감 (채택 — storyarc_snowy_queen.ts)
- **NPC**: 같은 마법 아카데미 동문. 힘이 폭주한 그날 옆에 있었지만 도망친 인물.
- **서사 축**: "그때 네 편에 서줬어야 했어." 사과를 전하러 가는 이야기.
- **톤**: 죄책감 ↔ 용서. 가족 아닌 책임감 기반.
- **복선 예**: 아카데미 문양, 실습복 조각, 과거 사고 때와 동일한 결정 패턴, 두 사람의 그림자 자국.
- **클라이맥스**: 두 개의 사과가 교차하며 성이 녹는다.

### 5. 유일한 목격자 (트라우마 축)
- **NPC**: 그녀의 힘이 처음 폭주한 날 얼어붙은 마을의 유일한 생존자(아이). 원망·공포·그리움이 뒤섞여 성장해 돌아옴.
- **서사 축**: "나는 너를 미워하러 온 건지, 보고 싶어서 온 건지 모르겠어."
- **톤**: 복잡한 양가감정. 용서가 쉽게 오지 않는다.
- **복선 예**: 어릴 적 장난감, 얼어붙은 가족 초상, 녹지 않는 발자국.
- **클라이맥스**: 처벌도 용서도 아닌, "살아있어 줘서 고마워"로 착지.

## 서사 축을 만드는 체크리스트

같은 보스에 대해 새로운 서사를 짤 때 고려할 것:

1. **NPC와 보스의 관계성**: 혈연 / 과거 동료 / 직업적 연관 / 피해자 / 조사자 / 무관계
2. **NPC의 동기**: 재회 / 증명 / 복수 / 사과 / 계약 이행 / 호기심
3. **탐험대가 이 여정을 시작한 표면 이유**: 소문 / 의뢰 / 조사 / 우연 / 개인사
4. **진실이 밝혀지는 방식**: 단서 조합 / 대화 / 회상 / 물리적 증거
5. **클라이맥스의 감정**: 재회 / 발견 / 선택 / 용서 / 수용
6. **복선 4개의 소재**: 장소에 남은 흔적 / NPC 몸에 나타나는 반응 / 보스 행동의 이상함 / 오브젝트

보스 설정(surface/true/motivation/twist)은 고정이어도, 위 6개 변수만 바꾸면 완전히 다른 이야기가 된다.

## 다른 보스에 응용할 때

- 꼭 "비운의 계승자" 같은 비극적 보스가 아니어도 됨. 코믹 보스(펭귄 대공 핀)에도 동일 프레임 적용 가능: NPC 관계성과 접근 동기만 바꿔도 다른 서사.
- 예: 펭귄 대공 → (a) 탐험대가 모피 상인이었을 때 / (b) NPC가 펭귄 학자일 때 / (c) 아카데미에서 우연히 만난 옛 친구 펭귄일 때 등.
